# 캡스톤 프로젝트 소개 웹 (EC2 + Docker, 또는 로컬)
본 저장소는 캡스톤 프로젝트 단일 소개 페이지와 API를 제공합니다.

## 1. 작품주제(Subject)
- 제목: 디지털 디톡스
- 요약: 스마트폰 중독 현상을 해결하기 위한 앱

## 2. 실용적 근거(Rationale)
- 문제: 최근 많은 사람들이 스마트폰 중독 현상을 겪고 있다.
- 근거(수치/설문/사례): 설문 결과 80%가 중독 현상을 겪었다.
- 기대 가치: 사용자들의 시간 활용성 증가, 중독 현상 해결

## 3. 핵심기능(Features)
- 기능 1: 휴대폰 전체, 앱별 잠금
- 기능 2: 앱의 하루 사용시간을 정하여 지정된 시간만큼 사용
- 기능 3: 캘린더 공유 기능으로 친구들과 일정공유

## 4. 구현환경(Environment)
- Front-End(프론트엔드): android student
- Back-End(백엔드): node.js, firebase
- Deploy(배포): google cloud

## 5. 팀원 구성 및 역할(Team)
- 우기현 - UI/UX 디자인, 프로젝트 개발
- 진건웅 - 프론트엔드, 안드로이드 개발
- 진건웅 - 백엔드, 자료 수집 및 발표

## 6. 실행 방법(Run)

```bash
docker compose up --build -d
# http://localhost:5000 < 프로젝트에 맞는 포트 또는 배포된 public IP >
```
